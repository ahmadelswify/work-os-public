# Weekly Review Workflow

A 15-30 minute session to reflect on progress and plan ahead.

## When to Use

User says: "weekly review", "what did I accomplish this week", "how am I tracking", or similar.

## Inputs Required

None — reads Active/ tasks, Archive/, and GOALS.md automatically.

## Workflow Steps

### Step 1: Review Completed Work
1. Scan Active/ for tasks with status `d` (done) from this week
2. Check Archive/ for recently moved items
3. List accomplishments grouped by goal alignment

### Step 2: Check Goal Progress
1. Read GOALS.md for current objectives
2. Map completed tasks to goals
3. Present goal health using indicators:
   - On track — regular progress this week
   - Needs attention — no tasks completed for this goal
   - At risk — blocked or significantly behind

### Step 3: Identify Blockers and Stale Items
1. Run `docs/garbage-collection/stale-task-sweep.md` rules
2. List blocked tasks with suggested unblocking actions
3. Flag tasks started but with no progress this week

### Step 4: Plan Next Week
1. Suggest top priorities based on P0/P1 tasks and goal alignment
2. Group as: Must do, Should do, If time allows
3. Flag upcoming deadlines

### Step 5: Suggest Maintenance
- Archive completed tasks
- Log significant accomplishments to Career/
- Update GOALS.md if priorities have shifted

## Success Criteria

- [ ] Accomplishments listed and celebrated
- [ ] Goal progress assessed
- [ ] Blockers and stale items identified
- [ ] Next week priorities suggested
- [ ] Maintenance actions proposed
